This repository collects resolutions of a number of open problems from the COLT open-problem proceeding, [commutative ring theory](https://www2.math.uconn.edu/~glaz/Publications_Selected%20Articles/OpenProblemsInCommutativeRingTheory.Springer14.pdf) and an open question in a FOCS 2022 paper by the authors.

The proof are generated from GPT-5.5 Pro via a simple prover verifier pipeline, and the paper is assembly by CLAUDE code, polished and verified by the authors.

We also formalize the solution to commutative ring theory in LEAN, using our agentic LEAN formalization pipeline.
